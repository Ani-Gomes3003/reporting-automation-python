# Reporting Automation System (Python)

## Overview
This project automates the generation of operational performance reports from raw Excel data using Python.

It eliminates manual Excel processing by automatically calculating productivity metrics, error rates, and performance summaries.

## Problem
Manual reporting requires repetitive Excel work:
- Calculating productivity manually
- Computing error rates
- Generating summary reports
- High risk of human error

## Solution
This Python automation script:

- Reads raw Excel input data
- Calculates productivity and error rate automatically
- Identifies low performers
- Generates structured Excel report
- Creates summary metrics automatically

## Tools Used
- Python
- pandas
- openpyxl
- Excel
